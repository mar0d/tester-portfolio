# Zgłoszenia błędów

Przykłady zgłoszeń błędów przygotowanych podczas testowania aplikacji webowej **Automation Exercise**.

Aplikacja: [Automation Exercise](https://automationexercise.com/?utm_source=chatgpt.com)

## Zakres testów

Zgłoszenia dotyczą między innymi:

* formularzy,
* koszyka zakupowego,
* produktów,
* komentarzy użytkowników,
* walidacji danych wejściowych.

## Zawartość zgłoszeń

Każde zgłoszenie błędu zawiera:

* tytuł błędu,
* opis problemu,
* kroki reprodukcji,
* oczekiwany rezultat,
* rzeczywisty rezultat,
* materiały pomocnicze (zrzuty ekranu, nagrania lub eksport z Jira).

## Lista zgłoszeń

| ID      | Tytuł                                                            |
| ------- | ---------------------------------------------------------------- |
| BUG-001 | Możliwość dodania produktu z ilością 0 do koszyka                |
| BUG-002 | Możliwość wysłania formularza kontaktowego bez treści wiadomości |
| BUG-003 | Nie można dodać komentarza do produktu                           |
